== Description ==

Alan Partridge hits WordPress! To show a random quote on your site just put <code>&lt;?php alanpartridge_quote(); ?&gt;</code> in your template.

== Installation ==

1. Upload `alanpartridgequotes.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php alanpartridge_quote(); ?>` in your templates, view screenshots.

== Screenshots ==

1. `<?php alanpartridge_quote(); ?>` has been placed in the wp-content/themes/default/sidebar.php file, just inside the `<div id="sidebar">` tag.
